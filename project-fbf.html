Fonctionnalités du site web

Le site Frame by Frame est un site vitrine immersif conçu pour valoriser une agence créative spécialisée dans le branding pour le cinéma. L'objectif est de montrer les services, la direction artistique et les réalisations de manière claire, sensible et percutante.

Fonctionnalités principales : Home page immersive, section Film (projets), page Contact.
Fonctionnalités secondaires : Newsletter, Blog, Shop.
Sur mobile, la hiérarchie est condensée pour conserver l'impact visuel tout en facilitant les interactions (navigation, repérage rapide, prise de contact).
